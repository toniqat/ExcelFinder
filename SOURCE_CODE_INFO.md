# ExcelFinder 소스코드 정보

## 프로젝트 개요

ExcelFinder는 여러 엑셀 파일에서 특정 값을 빠르게 검색할 수 있는 오픈소스 도구입니다.

## 소스코드 구조

### 주요 파일

- **main.py**: 애플리케이션 진입점
- **main_app.py**: 메인 애플리케이션 창 및 UI 로직
- **file_processor.py**: 엑셀 파일 처리 및 검색 로직
- **search_worker.py**: 병렬 검색 작업자 스레드
- **sheet_viewer.py**: 검색 결과 시트 뷰어 다이얼로그
- **search_exception_dialog.py**: 검색 예외 처리 설정 다이얼로그
- **config.py**: 설정 및 공통 임포트
- **build_exe.py**: 실행 파일 빌드 스크립트

### 지원 파일 형식

- **.xlsx**: Excel 2007+ 형식 (openpyxl 엔진 사용)
- **.xls**: Excel 97-2003 형식 (xlrd 엔진 사용)
- **.xlsm**: Excel 매크로 지원 형식 (openpyxl 엔진 사용)
- **.xlsb**: Excel 바이너리 형식 (pyxlsb 엔진 사용)

### 주요 기능

1. **다중 파일 검색**: 여러 엑셀 파일에서 동시 검색
2. **병렬 처리**: 최대 4개 스레드를 사용한 빠른 검색 (CPU 코어 수 제한)
3. **검색 모드**: 정확한 값 일치 또는 부분 일치 검색
4. **검색 예외 처리**: 특정 헤더 열 제외 및 조건부 제외
5. **결과 뷰어**: 검색 결과 행의 전체 데이터 표시
6. **오류 로깅**: 상세한 오류 정보 및 로그 파일 생성

## 빌드 방법

### 필요 조건

- Python 3.6 이상
- 필요한 패키지들 (requirements.txt 참조)

### 개발 환경 설정

1. **저장소 클론**:
   ```bash
   git clone <repository-url>
   cd ExcelFinder
   ```

2. **가상 환경 생성** (권장):
   ```bash
   python -m venv venv
   venv\Scripts\activate  # Windows
   # 또는
   source venv/bin/activate  # Linux/Mac
   ```

3. **의존성 설치**:
   ```bash
   pip install -r requirements.txt
   ```

4. **개발 실행**:
   ```bash
   python main.py
   ```

### 실행 파일 빌드

1. **자동 빌드** (권장):
   ```bash
   # 단일 실행 파일로 빌드
   python build_exe.py --type onefile
   
   # 디렉토리로 빌드 (빠른 시작)
   python build_exe.py --type onedir
   ```

2. **수동 빌드**:
   ```bash
   # PyInstaller 직접 사용
   pyinstaller --name=ExcelFinder --onefile --windowed --noupx --clean --noconfirm main.py
   ```

### 빌드 결과

- **onefile 모드**: `dist/ExcelFinder.exe` (단일 실행 파일)
- **onedir 모드**: `dist/ExcelFinder/` 폴더 (여러 파일)

## 기술적 세부사항

### 병렬 처리

- **기본값**: 최대 4개 스레드, CPU 코어 수를 넘지 않음
- **멀티프로세싱**: `multiprocessing` 모듈 사용
- **작업 분할**: 파일 단위로 작업을 분할하여 병렬 처리

### 메모리 최적화

- **캐싱**: 검색 결과의 행 데이터를 메모리에 캐싱
- **가비지 컬렉션**: 불필요한 객체의 적절한 해제

### 오류 처리

- **다단계 시도**: 여러 방법으로 파일 읽기 시도
